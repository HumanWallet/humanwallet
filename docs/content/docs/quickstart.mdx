# Quick Start Guide

Get HumanWallet integrated into your React application in just a few steps. This guide covers the essential setup to get you started quickly.

## Installation

Install the HumanWallet connector and required dependencies:

```bash
npm install @humanwallet/connector viem wagmi @tanstack/react-query
```

For development, you'll also need node polyfills:

```bash
npm install --save-dev vite-plugin-node-polyfills
```

## Environment Setup

Create a `.env.local` file with your ZeroDev Project ID:

```bash
VITE_ZERODEV_PROJECT_ID=your_project_id_here
```

## Basic Configuration

### 1. Configure Vite

Update `vite.config.ts`:

```typescript
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import { nodePolyfills } from 'vite-plugin-node-polyfills'

export default defineConfig({
  plugins: [react(), nodePolyfills()],
})
```

### 2. Setup Wagmi

Create `src/wagmi.ts`:

```typescript
import { createConfig, http } from 'wagmi'
import { sepolia, mainnet } from 'wagmi/chains'
import { humanWalletConnector } from '@humanwallet/connector'

export const config = createConfig({
  chains: [sepolia, mainnet],
  connectors: [
    humanWalletConnector({
      projectId: import.meta.env.VITE_ZERODEV_PROJECT_ID,
      appName: "My App",
    }),
  ],
  transports: {
    [sepolia.id]: http(),
    [mainnet.id]: http(),
  },
})
```

### 3. Add Providers

Update `src/main.tsx`:

```tsx
import React from 'react'
import ReactDOM from 'react-dom/client'
import { WagmiProvider } from 'wagmi'
import { QueryClient, QueryClientProvider } from '@tanstack/react-query'
import { config } from './wagmi'
import App from './App'

const queryClient = new QueryClient()

ReactDOM.createRoot(document.getElementById('root')!).render(
  <WagmiProvider config={config}>
    <QueryClientProvider client={queryClient}>
      <App />
    </QueryClientProvider>
  </WagmiProvider>
)
```

## Simple Integration

Here's a minimal component to get you started:

```tsx
// src/App.tsx
import { useConnect, useAccount, useDisconnect, useSignMessage } from 'wagmi'
import { useState } from 'react'

function App() {
  const { connectors, connect } = useConnect()
  const { address, isConnected } = useAccount()
  const { disconnect } = useDisconnect()
  const { signMessage } = useSignMessage()
  const [signature, setSignature] = useState<string>()

  const humanWallet = connectors.find(c => c.id === 'humanWallet')

  const handleSign = async () => {
    const sig = await signMessage({ message: 'Hello HumanWallet!' })
    setSignature(sig)
  }

  if (isConnected) {
    return (
      <div style={{ padding: '20px' }}>
        <h2>üéâ Connected!</h2>
        <p>Address: {address}</p>
        
        <button onClick={handleSign}>
          Sign Message
        </button>
        
        {signature && (
          <div>
            <h3>Signature:</h3>
            <code>{signature}</code>
          </div>
        )}
        
        <button onClick={() => disconnect()}>
          Disconnect
        </button>
      </div>
    )
  }

  return (
    <div style={{ padding: '20px' }}>
      <h1>üîê HumanWallet Demo</h1>
      
      <button onClick={() => connect({ connector: humanWallet })}>
        Connect with Passkey
      </button>
      
      <button onClick={() => (connect as any)({ 
        connector: humanWallet, 
        forceCreate: true 
      })}>
        Create New Wallet
      </button>
    </div>
  )
}

export default App
```

## Run Your App

Start your development server:

```bash
npm run dev
```

## Common Issues

### "Project ID not found"
- Check your `.env.local` file has the correct project ID
- Ensure you're using a valid ZeroDev project ID

### "WebAuthn not supported"
- Use HTTPS in production (localhost works for development)
- Ensure the browser supports WebAuthn (most modern browsers do)

### "Failed to create passkey"
- Check if the user cancelled the biometric prompt
- Verify your domain is configured correctly in ZeroDev
